# Wordpress Block External Services - وردپرس افزونه مسدودسازی سرویس‌های خارجی

[![WordPress Plugin Version](https://img.shields.io/badge/WordPress-6.0%2B-blue)](https://wordpress.org/)
[![License](https://img.shields.io/badge/License-GPL%20v2-green.svg)](https://www.gnu.org/licenses/gpl-2.0.html)

**Block External Services** is a lightweight WordPress plugin that blocks outgoing HTTP requests to specified domains including Google Fonts, Elementor, and Gravatar. It helps improve privacy, reduce external dependencies, and fix slow loading times when external services are blocked at the server level.

### Features
- 🚫 Block Google Fonts (fonts.googleapis.com & fonts.gstatic.com)
- 🚫 Block Elementor.com connections
- 🚫 Block Gravatar.com
- 🚫 Add any custom domain via admin panel
- ⚡ No AJAX - simple and fast
- 🎯 Removes DNS prefetch & preconnect hints
- 🔒 Blocks requests at multiple levels (HTTP output, enqueue, resource hints)

### Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings → Block External Services** to manage blocked domains

**افزونه مسدودسازی سرویس‌های خارجی** یک افزونه سبک وردپرس است که درخواست‌های خروجی به دامنه‌های مشخص شده (مانند فونت‌های گوگل، المنتور و Gravatar) را مسدود می‌کند. این افزونه به بهبود حریم خصوصی، کاهش وابستگی‌های خارجی و رفع مشکل کندی بارگذاری در سرورهایی که به سرویس‌های خارجی دسترسی ندارند، کمک می‌کند.

### قابلیت‌ها
- 🚫 مسدودسازی فونت‌های گوگل (fonts.googleapis.com و fonts.gstatic.com)
- 🚫 مسدودسازی ارتباطات المنتور (Elementor.com)
- 🚫 مسدودسازی Gravatar.com
- 🚫 امکان افزودن هر دامنه دلخواه از طریق پنل مدیریت
- ⚡ بدون Ajax - ساده و سریع
- 🎯 حذف تگ‌های DNS prefetch و preconnect
- 🔒 مسدودسازی در چند سطح (خروجی HTTP، enqueue، resource hints)

### نحوه نصب
1. پوشه افزونه را در مسیر `/wp-content/plugins/` آپلود کنید
2. افزونه را از طریق منوی «افزونه‌ها» در وردپرس فعال کنید
3. به **تنظیمات → مسدودسازی سرویس‌ها** بروید تا دامنه‌های مسدود شده را مدیریت کنید
